== Libraries

=== DenoVM

Execute JavaScript code in a secure Deno runtime from your Spring Boot application.

* link:denovm/index.html[DenoVM Documentation]

=== Flow

Spring Boot starter for Fynxt Lowcode Flow management. Auto-configures all Flow components when on the classpath.

* link:flow/index.html[Flow Documentation]

=== JWT

JWT token generation and validation library for Java/Spring. Provides JWT token generation and validation capabilities with configurable signing keys.

* link:jwt/index.html[JWT Documentation]
