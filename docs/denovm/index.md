# Fynxt DenoVM Library

Execute JavaScript code in a secure Deno runtime from your Spring Boot application.

## Overview

The Fynxt DenoVM library enables you to execute JavaScript code in a secure Deno runtime environment directly from your Spring Boot application. It provides a flexible execution model with support for both single-run and worker pool modes, making it suitable for both development and production environments.

### Key Features

* **Secure Execution**: Isolated Deno processes with configurable permissions
* **Flexible Execution Modes**: Single-run mode for development, worker pool mode for production
* **Rich SDK**: Built-in HTTP client, logger, and encoding utilities
* **Auto-Configuration**: Spring Boot auto-configuration support
* **Metadata Tracking**: Automatic logging and HTTP call tracking

See [Architecture](./architecture.md) for detailed information.
